# text2sentiment2osc

## Installation
1. Install pip (if you don't have it)
2. Install virtualenv (not absolutely nessecary, but recommended)
3. `virtualenv venv`
4. `. venv/bin/activate`
5. `pip install requirements.txt`
6. `export FLASK_APP=app.py`

## Run it
1. `. venv/bin/activate` (if virtualenv isn't active)
3. `flask run`

## Kill it
1. Ctrl + C
2. `deactivate`
